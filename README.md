# CBT Platform - Computer-Based Testing System

A comprehensive, secure, and modern platform for conducting computer-based examinations in educational institutions.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open browser to http://localhost:5173/login
```

## ✨ Features

- ✅ **5 User Roles**: Student, Teacher, Invigilator, School Admin, Super Admin
- ✅ **Beautiful UI**: Professional design with your CSS variables
- ✅ **Secure Authentication**: Role-based access control
- ✅ **Dashboard**: Sidebar + Topbar + Role-specific content
- ✅ **Demo Mode**: Test without backend
- ✅ **Responsive**: Works on all devices

## 🎨 Design

- **Primary Color**: #167d1c (Green)
- **Accent Color**: #ff9001 (Orange)
- **Professional**: Classy, modern design
- **Consistent**: Uses CSS variables throughout
